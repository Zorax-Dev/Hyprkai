<h2 id="installation">
  <a href="https://git.io/typing-svg"><img src="https://readme-typing-svg.herokuapp.com?font=Lexend&weight=900&size=28&pause=1000&color=9B3BBC&width=435&height=41&lines=INSTALLATION+GUIDE" alt="Typing SVG" /></a> 
</h2>

- The installation script is designed for a minimal Arch Linux install, but may work on some Arch-based distros. 

- To install Hyprkai, execute following commands in Your terminal / konsole
```
sudo pacman -S --needed git base-devel
git clone https://github.com/Zorax-Dev/Hyprkai.git
cd Hyprkai
chmod +x install.sh
./install.sh
```
> [!TIP]
> Install script auto download yay if not found however we reccomend to download yay before running installation

- **YOU WILL FIND ALL KEYBINDS IN hypr/config/binds.confconf**
- Some Imp KEYBINDS 

| Keybind | Action |
|--------|--------|
| Super + ` | Open terminal | 
| Super + Q | Close window | 
| Super + A | App launcher | 
| Super + E | File manager |
| Super + L | System launcher | 
| Super + B | Waybar Switcher |
| Super + Shift + S | Screenshot |
